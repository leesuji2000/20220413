#컴퓨터구조를 왜 학습해야 할까? 
-현대컴퓨터는 어떤 기계인가
-상태머신과 소프트 웨어
-우리가 이루려는 목표, 고성능, 저전력
-결국은 도메인의 문제

#컴퓨터의 이해
하드웨어와 소프트웨어의 결합체
소프트웨어 -> 시스템 소프트웨어(OS), 응용 소프트웨어(app)
하드웨어 -> 입력장치(키보드, 마우스, 플로피 디스크 판독기, 자기 테이프 구동장치) -> 처리장치 : 주기억장치->보조기억장치(자기디스크, 메모리)// 중앙처리장치(제어장치, 연산장치)

#컴퓨터 하드웨어의 구성
IO장치, 저장소, 프로세싱

#데이터의 표현
-정보
	-어떠 사물에 대한 소식이나 자료
	-가공된 데이터
-데이터
	-정보를 작성하기 위해 필요한 자료나 정보를 처리하거나 전송할때, 이진(binary)이나 디지털과 같은 좀 더 편리한 형태로 바뀌어진 자료
	-정보의 원재료
-데이터는 날것, 데이터를 가공하면 정보가 된다.

#데이터 종류
-수치데이터(numerical data): 연산용 데이터
-비수치데이터(alchanumerical data): 입출력용 데이터(나이, 이름 같이 표시하는거)
-특수문자(special character): 입출력/연산용 데이터(보안, 암호할 떄 사용하는거)

#단위
-비트Bit: 0, 1
-바이트Byte: 8Bit
-위드Word: 기계에 따라 상이함, 1word = 32 bit or 64bit
-KB: 1024 bytes
-GB: 1024 MB = 2^20byte
-TB: 1024 GB =2^30byte

#컴퓨터 구조를 배우는 이유는 한정적인 재화(자료, 소스)를 효율적으로 사용하기 위함이다.

#컴퓨터 구조로 본 패러다임의 변화
-컴퓨터의 시작 -> 사무자동화 기계로 인식 -> 네트위크의 단말로 통신기기의 열풍 -> 컴퓨터의 미래: 유비쿼터스, 모바일&wireless, IOT, smart computer(machine learning, deep learning, AI)

#컴퓨터의 태동
주판 -> 파스칼 계산기 -> 라이프니츠의 계산기 -> 찰스 배비지 차분기관(**전기가 없지만 기계 설비 완료) -> 에이컨의 MARK-1 -> 머클리&에커드의 에니악(폰노이만 구조-데이터와 시스템의 자동화/진공관 사용) -> 스마트폰 -> AI

#사무자동화 기계로 인식하던 시대
-사무 자동화 기기 정보처리 유형에 따른 분류
	-자료준비기기: 워드 프로세서, 복사기, 프린터
	-자료처리기기: 워드 프로세서, 개인용 컴퓨터
	-자료전송기기: 팩스, 이메일, 원격 회의 시스템
	-자료저장기기: 마이크로 필름, com-> car, 광디스크 -> 하드디스크

#네트워크의 단말(터미널 = 개개인 컴퓨터= dege)시대
-acess net: 가장 지엽적인 인터넷 회전 지원 회사
-ISP A, B, C :  internet service provider 다국적 기업(skt, 통신사)
-인터넷 세계관에 의하면 인터넷 말은 edge(주변부)와 core(중심부)로 구성
-IXP(internet exchange point): 여러 통신사를 연결해주는 곳
-regional net: 지협적인 곳에 연결해주는(필요없긴 한데 빠르게 특화된 분야에 사용)

#컴퓨터의 미래 예측
-유비쿼터스 시대의 도래: 	
	-유비쿼터스 + 컴퓨팅 -> 가상 세계에서 생활이 가능
	-실세계의 매핑 -> 가상 세계 -> 초연결에 의한 개념 도입
-모바일과 wireless 전성시대
	-보안 중요해짐
	-자이로스코프를 통해서 내가 어디에 있는지 스마트폰을 알 수 있다.
	-유선 사업자와 스마트폰 사업자의 위상변화(유선회사 -> 무선회사(삼성, 애플))
-IOT
-모바일의 근간이 된다
	-IOT(테이터생성) -> Cloud(데이터 저장) -> big data -> service,,
-smart computer(machine learning)
	-인공지능이 컴퓨터를 관리
