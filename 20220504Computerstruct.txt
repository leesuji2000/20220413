#컴퓨터 시스템의 이해
#컴퓨터 구성요소의 인지와 기능의 조합
	-중앙처리장치(central processing unit)
		-cpu/mpu(병렬처리로 cpu여러개 묶어놓은거)
		-사물 인터넷 디바이스 h/w 플랫폼 종류
	-주변장치(peripheral device)
		-기억장치
		-보조기억장치

#cpu
	-마더보드: 데이터의 전달 통로가 디자인 되어있는 메인보드
	-하나의 칩으로 이루어짐(산술논리연산장치ALU와 제어장치CU로 구성)
	-실행 프로그램의 명령 해석, 실행, 장치 제어, ALU, CU, 각종 레지스터로 구성

#mpu
	-cpu를 LSI(고밀도 집적회로)화 한 일종의 통합장치
	-CISC(complex instruction set computer)
	-RISC(reduced instruction set computer)
	-bit slice MPU등이 존재한다

cpu가 발전해서 -> 사물인터넷 디바이스 h/w 플랫폼 종류: 아두이노 탄생(대표적인 오픈소스 h/w 플랫폼, atmel사의 avr이나 coretex-m3를 탑재한 마이크로 컨트롤러 보드), 라즈베리 파이, 갈리에오, 에디슨 등 오픈형 h/w가 있다
=> 집에 있는 냉장고 같은 것들에 특화된 cpu

#기억장치(memory unit)
	-RAM(random acess memory)
		-DRAM(dynamic RAM)
		-SRAM(static RAM)
	-ROM(read only memory)

#주기억장치와 보조기억장치의 관계
1. ROM -> CPU(전원 부팅 시 cpu는 자동으로 rom 에 있는 프로그램 실행)
2. 보조기억장치 -> RAM 로 적재시킨다.(load)
3. RAM <-> CPU (데이터를 주고 받음)
4. RAM -> 보조기억 장치(저장)

#보조기억장치
	-플래시 기억장치 : EEPROM의 한 종류(RAM과 ROM의 중간위치)
	-CF(compact flash)메모리
	-SSD(solid state drive)

#참고상항
-디스크의 성능과 파라미터
	-헤드를 해당 트랙으로 이동: 탐색 시간
	-데이터가 포함된 섹터가 회전되어 헤드 아래로 올때까지 대기: 회전 지연
	-데이터 전송: 데이터 전송 시간
-데이터 접근 시간
	-데스크 접근 시간 = 탐색 시간 + 회전 지연  + 데이터 전송 시간

-비지니스 환경에서의 통신과 네트워킹의 역할
-양자 컴퓨터
-웹의 진화 - 글로벌 인터넷

#비지니스 환경에서 통신과 네트워킹의 역할(추세)
	-80년대 이후 지속되고 있는 급속한 정보통신기술의 발전으로 그에 따른 컴퓨터 구조의 변화도 비약적인 발전과 새로운 시장 구도를 변화시키고 있는 상황
	-NOS(network operating system)의 역할 확대 => 컴퓨터 내부의 정보교환이 중요해짐
	-최근 네트워크 장비가 미,중 무역전쟁의 핵심 쟁점 부각
	-유, 무선, 통신, 방송 및 통신, 컴퓨터의 융합 등 컨버전스 가능
	-소프트웨어 기술과 고밀도 직접 기술의 발달 -> 신형의 서비스와 장비 등장
	-사용자 중심의 신기술이 활발하게 개발됨

#양자컴퓨터
	-중첩(superposition)과 얽힘(entanglement)등 양자의 고유한 물리학적 특성을 이용하여, 다수의 정보를 동시 처리할 수 있는 새로운 개념의 컴퓨터
	-현대 반도체 칩의 미세회로에서 발생하는 누설 전류로 인한 고전 컴퓨터 성능 한계 돌파를 위한 대안
	-양자적 정보 단윈인 양자비트, 큐비트를 정보처리의 기본 단위로 하는 양자 병렬처리를 통해 정보처리 및 연산속도 해결
